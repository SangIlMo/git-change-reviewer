# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-02

### Added
- Initial release of PR Change Annotation plugin
- `/annotate-pr` slash command for Claude Code
- Context-based, inferred, and conflict annotation types
- Dry-run mode for preview without posting
- Large PR support with batched review posting
- Auto-detect current branch PR when no number given
- Helper scripts: fetch-context.sh, parse-hunks.sh, post-review.sh
